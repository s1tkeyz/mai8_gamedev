# Программирование игр

Проект по курсу "Програмирование игр" 2026

Выполнил: ст-т Даутов Т. Б. (М8О-406Б-22)

## Сеттинг
Игрок - инженер на орбитальной станции, привязанный упругим тросом к базовому модулю.
В реальном времени он должен перемещаться в невесомости, поддерживая работоспособность станции.

## Механики
1. Упругий трос-лебёдка: управляемая физическая связь для навигации и воздействия на объекты в невесомости.
(реализовано с помощью Rigidbody и LineRenderer)

2. Диагностический сканер: показывает каркас станции, эффект рентгена (реазизовано с помощью URP Shader) при удерживании  клавиши R.

3. Система событий для состояния модулей: случайная поломка модулей и визуальное отображение, UI + возможность починки (на определенном расстоянии на клавишу F).

Управление игрока - Unity.InputSystem, UI стандартный Canvas
